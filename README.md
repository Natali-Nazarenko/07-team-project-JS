# JS-team07-project

Цей проект було створено за допомогою Vite. 
Для знайомства перейдіть за посиланням: 
https://natali-nazarenko.github.io/07-team-project-JS/

## Задачі проекту

Цей сайт був створенний для комфортного ознайомлення меломанів з їх улюбленими гуртами і співакми. Дає можливість подивитись на YouTube відео їх улюлених пісень. Знайти інформацію про музиканта. Або за своїми уподобаннями в жанрі музики, відкрити для себе творчість когось нового.

## Використані технології

- HTML
- CSS
- JavaScript
- Git
- bundler Vite
- Node Package Manager(npm)
- бібліотек: "axios", "izitoast", "swiper", "css-star-rating"
- запроси на БД виконанні за допомогою https://sound-wave.b.goit.study/api-docs/#/

## Запуск/розгортання проекту

1. щоб подивитись і протестувати проект, вам достатньо перейти по цьому посиланню:
https://natali-nazarenko.github.io/07-team-project-JS/

2. для розгортання проекту в себе на компьютері, треба виконати декілька команд:
    - скопіювати код проекту - https://github.com/Natali-Nazarenko/07-team-project-JS.git
    - в себе на компьютері набрати команду: git clone "посилання на проект"
    - інсталювати всі залежності: npm install
    - і запустити зборщик Vite: npm run dev
